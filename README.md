# To-Do List

A modern and responsive To-Do List application built with React.

## Requisitos del Proyecto

### Tecnologías

- **React**: Framework principal para construir la interfaz de usuario
- **Tailwind CSS**: Framework de utilidades CSS para el diseño y estilos

### Requisitos Previos

Antes de comenzar, asegúrate de tener instalado:

- Node.js (versión 14 o superior)
- npm o yarn

### Instalación

1. Clona el repositorio:
```bash
git clone git@github.com:Crisca-cyver/To-Do.git
cd To-Do
```

2. Instala las dependencias:
```bash
npm install
```

3. Inicia el servidor de desarrollo:
```bash
npm start
```

### Funcionalidades

- Agregar nuevas tareas
- Marcar tareas como completadas
- Eliminar tareas
- Interfaz responsive y moderna

## Desarrollo

Este proyecto fue creado con las siguientes herramientas:
- React para la lógica de componentes y gestión de estado
- Tailwind CSS para estilos rápidos y personalizables
